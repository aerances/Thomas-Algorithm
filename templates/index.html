Thomas Algorithm (Tridiagonal Matrix Solver)

1. Mathematical Discussion

The Thomas Algorithm is an optimized variant of Gaussian elimination engineered to solve tridiagonal linear equation systems of the shape \(Mx = r\). A tridiagonal matrix contains non-zero entries exclusively on the main diagonal (\(b_i\)), the sub-diagonal (\(a_i\)), and the super-diagonal (\(c_i\)).

$$M = \begin{pmatrix} b_1 & c_1 & 0 & \dots & 0 \\ a_2 & b_2 & c_2 & \dots & 0 \\ 0 & a_3 & b_3 & \dots & 0 \\ \vdots & \vdots & \vdots & \ddots & c_{n-1} \\ 0 & 0 & 0 & a_n & b_n \end{pmatrix}$$

The system is numerically stable if it satisfies the condition of strict diagonal dominance:

$$\lvert b_i \rvert \geq \lvert a_i \rvert + \lvert c_i \rvert \quad \text{for all } i$$

2. Step-by-Step Worked Examples (Static Reference)

Example 1: Symmetric Dominant System

Consider a \(3 \times 3\) system where:

$$b = [2, 2, 2], \quad a = [0, -1, -1], \quad c = [-1, -1, 0], \quad r = [1, 0, 1]$$ Forward Sweep:
  • Row 1: \(\gamma_1 = \frac{-1}{2} = -0.5\), \(\rho_1 = \frac{1}{2} = 0.5\)
  • Row 2: \(\gamma_2 = \frac{-1}{2 - (-1)(-0.5)} = -0.6667\), \(\rho_2 = \frac{0 - (-1)(0.5)}{1.5} = 0.3333\)
  • Row 3: \(\rho_3 = \frac{1 - (-1)(0.3333)}{2 - (-1)(-0.6667)} = \frac{1.3333}{1.3333} = 1.0\)
Backward Substitution:
  • \(x_3 = \rho_3 = 1.0000\)
  • \(x_2 = 0.3333 - (-0.6667)(1.0) = 1.0000\)
  • \(x_1 = 0.5 - (-0.5)(1.0) = 1.0000\)

Solution Set: \(X = [1.00, 1.00, 1.00]\)

Example 2: Non-Symmetric System

Consider a \(3 \times 3\) system where:

$$b = [4, 4, 4], \quad a = [0, -1, -2], \quad c = [-2, -1, 0], \quad r = [2, 5, 2]$$ Forward Sweep:
  • Row 1: \(\gamma_1 = \frac{-2}{4} = -0.5\), \(\rho_1 = \frac{2}{4} = 0.5\)
  • Row 2: \(\gamma_2 = \frac{-1}{4 - (-1)(-0.5)} = -0.2857\), \(\rho_2 = \frac{5 - (-1)(0.5)}{3.5} = 1.5714\)
  • Row 3: \(\rho_3 = \frac{2 - (-2)(1.5714)}{4 - (-2)(-0.2857)} = \frac{5.1428}{3.4286} = 1.5000\)
Backward Substitution:
  • \(x_3 = 1.5000\)
  • \(x_2 = 1.5714 - (-0.2857)(1.5) = 2.0000\)
  • \(x_1 = 0.5 - (-0.5)(2.0) = 1.5000\)

Solution Set: \(X = [1.50, 2.00, 1.50]\)
