Thomas Algorithm (Tridiagonal Matrix Solver)

1. Mathematical Discussion

The Thomas Algorithm is an optimized variant of Gaussian elimination engineered to solve tridiagonal linear equation systems [cite: 116, 119, 120] of the shape \(Mx = r\)[cite: 120]. A tridiagonal matrix contains non-zero entries exclusively on the main diagonal, the sub-diagonal below it, and the super-diagonal above it. It can be parameterized cleanly using three vectors: \(a_i, b_i,\) and \(c_i\).

$$M = \begin{pmatrix} b_1 & c_1 & 0 & \dots & 0 \\ a_2 & b_2 & c_2 & \dots & 0 \\ 0 & a_3 & b_3 & \dots & 0 \\ \vdots & \vdots & \vdots & \ddots & c_{n-1} \\ 0 & 0 & 0 & a_n & b_n \end{pmatrix}$$

Algorithmic Logic Mechanics

  • Stage 1: Forward Sweep: Eliminates sub-diagonal values (\(a_i\)) progressively to convert the equation framework into an upper-unit triangular format defined by updated vectors \(\gamma_i\) and \(\rho_i\).
  • Stage 2: Backward Substitution: Works in reverse from the final element (\(x_n = \rho_n\)) back to \(x_1\) to derive the exact solution coordinates swiftly[cite: 122, 261, 263].

Algorithmic Stability Criteria

The method is mathematically stable against division error and tracking deviation if the system is diagonally dominant[cite: 296, 298]:

$$\lvert b_i \rvert > \lvert a_i \rvert + \lvert c_i \rvert \quad \text{for all } i$$

2. Step-by-Step Worked Example (Static Content)

Consider the system where:

$$b = [2, 2, 2], \quad a = [0, -1, -1], \quad c = [-1, -1, 0], \quad r = [1, 0, 1]$$

Step 1: Row 1 Calculations

\(\gamma_1 = \frac{c_1}{b_1} = \frac{-1}{2} = -0.50\)
\(\rho_1 = \frac{r_1}{b_1} = \frac{1}{2} = 0.50\)

Step 2: Row 2 Transformations

\(\gamma_2 = \frac{c_2}{b_2 - a_2\gamma_1} = \frac{-1}{2 - (-1 \times -0.5)} = \frac{-1}{1.5} = -0.67\)
\(\rho_2 = \frac{r_2 - a_2\rho_1}{b_2 - a_2\gamma_1} = \frac{0 - (-1 \times 0.5)}{1.5} = \frac{0.5}{1.5} = 0.33\)

Step 3: Row 3 (Final Row) Transformations

\(\rho_3 = \frac{r_3 - a_3\rho_2}{b_3 - a_3\gamma_2} = \frac{1 - (-1 \times 0.33)}{2 - (-1 \times -0.67)} = \frac{1.33}{1.33} = 1.00\)

Step 4: Backward Substitution Sweep

\(x_3 = \rho_3 = 1.00\)
\(x_2 = \rho_2 - \gamma_2 x_3 = 0.33 - (-0.67 \times 1.00) = 1.00\)
\(x_1 = \rho_1 - \gamma_1 x_2 = 0.50 - (-0.50 \times 1.00) = 1.00\)

Final Solution Set: \(X = [1.00, 1.00, 1.00]\)
